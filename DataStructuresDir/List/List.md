# List

## **What** is a List?
* A **List** is a data structure that contains a linear sequence of a number of elements that have the same type.

* The items can be of any data type
    * Numbers (1, 2, 10, 8, 12).
    * Characters ('H', 'O', 'L', 'A').
    * Cities ("MVD", "BSAS", "NY").

* We need to define the order of the items.
